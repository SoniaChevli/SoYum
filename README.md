# SoYum
Full stack web application to share pictures of food and easily discover new and trendy restaurants. 

![Alt text](./SoYum_Screenshot_10-6.png)

## Getting Started
### Installing Dependencies
From within the root directory:

```
npm install
nodemon
```
In new terminal tab:
```
mongod
```
